# Portfolio

Build repo for [antoniodesign.work](https://www.antoniodesign.work) — a static Astro site. Positioning infrastructure, not a showcase.

Thinking, scoping, and drafts live in `~/Vault/portfolio/`. Add with `/add-dir ~/Vault/portfolio` for session context — it holds the voice, handoff rules, and current state.

## Principle

Restraint beats features. Any refactor earns its place.

## Content destinations

- Articles → `src/content/articles/<slug>.md`
- Project case studies → `src/content/projects/<slug>.md`
- Schema: `src/content.config.ts`
